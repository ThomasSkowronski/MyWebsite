Dice Game

I made this dice game using Bootstrap and JQuery. It is a simple game where the player tries to get a higher score by rolling the dice.
If too many 1's are rolled then the round ends with a score of 0.
I made this webpage to see how the game plays out over a number of turns with different amounts of dice and strikes in the game.

Weight Tracker Android App

This Android app allows the user to set a goal for weight loss and track their progress over time.
The app saves the data about the user with shared preferences and weight entries can be saved along with a picture in order to see progress.
An SQLite database handles the entries and these can be viewed in chronological order and deleted as well.
